# Stream Slicer

Thumbnail generator for Twitch/YouTube stream covers.

## Deploy in 3 steps

### 1. Install dependencies
```
npm install
```

### 2. Build
```
npm run build
```
This creates a `dist/` folder.

### 3. Host
Drag the `dist/` folder to https://app.netlify.com/drop — you'll get a live URL immediately.

## Local dev
```
npm run dev
```
Opens at http://localhost:5173
